# Doctor Management System (PHP)

## 🔹 Description
This is a simple Doctor Management System built using PHP and MySQL. 
It helps manage doctors, patients, and appointments in an organized way.

## 🔹 Features
- Add and manage doctors
- Manage patient records
- Handle appointments
- Simple and user-friendly interface

## 🔹 Tech Stack
- PHP
- MySQL
- HTML/CSS

## 🔹 Project Structure
- index.php → Main page
- doctors.php → Doctor management
- patients.php → Patient management
- appointments.php → Appointment handling
- config.php → Database connection

## 🔹 What I Learned
- Backend development using PHP
- Database connectivity
- CRUD operations
- Project structuring

## 🔹 How to Run
1. Install XAMPP / WAMP
2. Place project in htdocs folder
3. Start Apache & MySQL
4. Open browser and run:
   http://localhost/project-folder-name
